🔧 常见问题排查

问题1：Vercel 部署失败

症状：PR 无法自动部署到 Vercel

排查：

  1. 检查 Vercel Dashboard 中的错误日志
  2. 检查构建日志
  3. 确认环境变量配置正确
  4. 检查依赖版本兼容性

问题2：Preview 环境与本地不一致

症状：Preview 环境显示与本地不同

排查：

  1. 清除浏览器缓存
  2. 检查环境变量配置
  3. 检查 API 配置
  4. 对比构建输出

问题3：生产环境性能下降

症状：部署后性能不如预期

排查：

  1. 检查 Vercel Analytics
  2. 运行 Lighthouse 测试
  3. 检查网络请求
  4. 优化图片和资源
