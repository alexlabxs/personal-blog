## 🔧 常见问题排查

### 问题1：页面无法访问
**症状**：访问 http://localhost:3000 显示错误
**排查**：
```bash
# 检查开发服务器是否运行
ps aux | grep "next dev"

# 检查端口是否被占用
lsof -i :3000

# 如果端口被占用，杀掉进程
kill -9 <PID>

# 重新启动开发服务器
npm run dev
```

### 问题2：样式显示异常
**症状**：页面样式不对，看起来很丑
**排查**：
- 检查是否正确加载了 Tailwind CSS
- 检查是否正确加载了自定义 CSS
- 清除浏览器缓存并刷新

### 问题3：TypeScript 错误
**症状**：代码中有类型错误
**排查**：
```bash
# 检查 TypeScript 错误
npx tsc --noEmit

# 修复所有类型错误后重新启动
npm run dev
```

### 问题4：语言切换不工作
**症状**：切换语言后文字不变
**排查**：
- 检查 URL 是否正确更新
- 检查浏览器是否有缓存
- 检查字典文件是否正确
