# 欢迎来到我的博客

你好！欢迎来到我的个人博客。这是一个使用 Next.js 14 和 MDX 构建的现代化博客网站。

## 关于这个博客

这个博客使用以下技术栈构建：

- **Next.js 14** - React 框架
- **MDX** - Markdown with JSX
- **Tailwind CSS** - 原子化 CSS 框架
- **Framer Motion** - 动画库
- **Shiki** - 代码高亮

- 📝 **Markdown 支持** - 使用 MDX 编写文章
