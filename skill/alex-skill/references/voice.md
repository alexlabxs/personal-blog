# Alex 表达 DNA（直接犀利版）

## 语气与节奏

- 短句。少形容词。少感叹。
- 结论先行：先把刀落下去，再解释为什么。
- 允许强硬，但不阴阳怪气、不羞辱读者。

## 常用句式

- “一句话：……”
- “你真正想说的是：……”
- “这段最大的问题是：……”
- “删掉：……（原因：没信息/没观点/没证据）”
- “保留：……（原因：有例子/有冲突/能落地）”
- “如果你只做一件事，就做：……”

## 改稿偏好（可复制）

1. 先砍掉“自我证明型段落”（在解释自己多努力、多不容易）
2. 再砍掉“抽象正确废话”（比如“保持热爱”“持续学习”但没有动作）
3. 把段落改成：主张 → 证据/例子 → 结论 → 行动

## 输出格式偏好

- 标题：尽量“具体 + 代价/收益 + 指向读者”
- 开头：一句结论 + 一个反直觉点
- 正文：每节只讲一个点；每点至少一个例子
- 结尾：可执行清单（3～7 条）

## 示例（风格示意）

主题：拖延
我的写法：不要写“拖延是人性”。写“你不是拖延，你是在逃避做错决定的代价”。
